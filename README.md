# 🛒 Amazon Clone

An Amazon-inspired full-stack e-commerce web app built with **React, Firebase, and Stripe (Payment Integration)**.  
This project replicates core Amazon functionalities like browsing products, adding to cart, checkout, and payment flow.

---

## 🔗 Live Demo

- **Hosted Link (Fully Working)**: [Amazon Clone (Hosting)](https://clone-98b3f.web.app)
- **Live Link (Firebase Functions + Stripe Integration)**: [Amazon Clone (Live)](https://us-central1-clone-98b3f.cloudfunctions.net/api)
))  
  *(⚠️ Note: Stripe payment flow works perfectly in the local environment (npm run dev) using test cards.
On the hosted version, it’s under improvement due to Firebase Functions deployment config.)*

---

## 🚀 Tech Stack

- **Frontend:** React, Context API, React Router, Tailwind CSS  
- **Backend:** Firebase Functions, Node.js, Express  
- **Database & Auth:** Firebase Firestore, Firebase Authentication  
- **Payments:** Stripe (Test Mode Integration)  

---

## ✨ Features

✔️ User Authentication (Sign Up / Login)  
✔️ Add to Cart & Checkout  
✔️ Stripe Payment Integration (Test Mode)  
✔️ Order History (Stored in Firebase)  
✔️ Fully Responsive UI
